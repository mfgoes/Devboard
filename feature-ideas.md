# DevBoard – Feature Ideas

## High Impact Features

### Command Palette
`Ctrl+K / Cmd+K` to open a searchable command palette.

Examples:
- create sticky
- create connector
- create rectangle
- export PNG
- zoom to fit
- duplicate selection

Benefits:
- faster workflow
- discoverable commands
- easy to extend later


### Quick Create Gestures
Speed up creation without switching tools.

Examples:
- Double click canvas → create sticky
- Hold **T** + click → text node
- Hold **S** + drag → section
- Drag from sticky edge → auto connector


### Smart Connectors
Upgrade connectors to be more diagram-friendly.

Ideas:
- Attach to edges instead of center
- Snap to anchor points
- Auto-reroute around nodes
- Optional orthogonal connectors

Connector styles:
- arrow
- straight line
- curved
- dotted

Arrow heads:
- arrow
- flat
- circle
- none


### Node Grouping
Allow grouping multiple nodes.

Features:
- multi-select nodes
- `G` to group
- drag group together
- optional collapsible groups
- frame-style container


### Keyboard First Workflow

Suggested shortcuts:

| Key | Action |
|----|----|
| N | Sticky |
| T | Text |
| L | Connector |
| R | Rectangle |
| E | Ellipse |
| Cmd/Ctrl + D | Duplicate |
| Cmd/Ctrl + G | Group |
| Cmd/Ctrl + Z | Undo |
| Shift + Drag | Multi-select |

# Polish Features

### Snap + Alignment Guides
While dragging:
- snap to node edges
- snap to centers
- snap to grid

Show alignment guides like design tools.


### Zoom To Fit
Shortcut: `Shift + 1`

Automatically frame all nodes on screen.


### Sticky Auto Resize
Sticky notes resize automatically as text grows while maintaining padding.


### Canvas Search
`Cmd/Ctrl + F`

Search for text inside nodes and zoom to results.
