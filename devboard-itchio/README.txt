DevBoard — Offline Whiteboard for Solo Devs
============================================

DOWNLOAD & INSTALL
------------------
1. Extract the html zip to any folder on your computer.
2. Open index.html in your browser (Chrome, Firefox, Edge, or Safari).
   - Double-click the file, or drag it into your browser window.
   - Keep index.html and the stickers/ folder together in the same directory.

No install required. No internet connection needed. Works fully offline.

USAGE TIPS
----------
- Pan:        Space + drag  |  Middle-mouse drag  |  Two-finger touchpad scroll
- Zoom:       Scroll wheel  |  Pinch gesture (trackpad)
- New sticky: Press S, then click on the canvas
- New shape:  Press R, then click and drag
- Text block: Press T, then click and drag
- Connector:  Press L, then click an anchor dot on any node
- Select:     Press V (or click the arrow in the toolbar)
- Delete:     Backspace or Delete key (with node selected)
- Copy/Paste: Cmd/Ctrl+C  |  Cmd/Ctrl+V  |  Cmd/Ctrl+D (duplicate)

EXPORT & SHARE
--------------
- Export PNG:   TopBar → "Export PNG"
- Save JSON:    TopBar → "Save" — downloads a .json board file
- Load JSON:    TopBar → "Load" — restores a saved board
- Share Link:   TopBar → "Share" — copies a URL with the board encoded in it

Your board auto-saves to localStorage between sessions.

BROWSER NOTES
-------------
- Brave users: Shields may block localStorage. Click the Shields icon
  and set "Block fingerprinting" to "Standard" for this page.
- File:// access is required for stickers to load. If stickers appear
  broken, try a local server: python3 -m http.server 8080
