# Tech Stack
This document covers all technologies used in DevBoard.
## Frontend
- **React 18** + TypeScript — UI layer
- **react-konva** — canvas rendering
- **Zustand** — state management (localStorage persistence)
- **Tailwind CSS v3** — UI chrome
## Build
- **Vite** + vite-plugin-singlefile — produces a single dist/index.html
