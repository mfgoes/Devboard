All features

Everything on one canvas

Built for the way developers actually think — not for slideware or stakeholder decks.

Infinite canvas

Pan with spacebar + drag or middle mouse. Pinch-to-zoom. Dot grid tracks your position. Re-center anytime.

Sticky notes

Place, drag, resize, double-click to edit. Eight color swatches. Copy, paste, duplicate — or Alt+drag for instant copies.

Smart connectors

Bezier arrows snap to anchor points and follow nodes when you move them. Straight and orthogonal styles too.

Shapes

Rectangles, ellipses, diamonds, triangles. Each takes a text label and connects to any other node.

Text blocks

Standalone text with bold, italic, underline, bullet lists, links, alignment, and color. No background in the way.

Code blocks

Drop syntax-highlighted code snippets onto the canvas. Reference the exact function you're designing around.

Undo / redo

Full undo/redo stack. Alignment guides appear while dragging near other nodes' edges or centers.

Groups & sections

Group nodes with ⌘G so they move together. Section labels mark regions. Lock nodes to freeze layout elements.

Export & share

Export PNG. Save/load JSON. Generate a shareable link with the full board encoded in the URL hash — no server.

Themes

Dark mode by default. Light mode for the rest of the team. Multiple color schemes.

Images

Drop PNG or JPG files directly onto the canvas. Works offline. Load a whole workspace folder for multi-file projects.

Stickers

Drop emoji stickers on the canvas for quick visual markers, status labels, or just vibes.
