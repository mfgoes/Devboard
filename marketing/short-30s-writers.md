# 30s Spot — Writers & worldbuilders

**Goal:** speak directly to novelists, TTRPG GMs, screenwriters, worldbuilders. Lean on `[[wikilinks]]` + canvas-as-plot-board.
**Best for:** TikTok #writingcommunity, #worldbuilding, #dndnotes, Bluesky writer crowd.
**Tone:** "I built this because I was tired of [their pain]." First-person voiceover OK.

---

## Hook options (pick one, A/B test)

- **A:** "I write fantasy. Every notes app made me pick: link my characters, OR draw the relationship map. Not both. So I built one that does both."
- **B:** "Every writer I know has a folder of `.md` files AND a Miro board AND a Notion page. I made the app that's just one workspace."
- **C:** *(no voiceover, on-screen only)* "POV: your character notes know about your plot map."

## Beats

| Time | What you show | Voiceover / overlay |
|---|---|---|
| 0–3s | Open on a `Character — Elara.md` doc. Type `[[Kingdom of Vael]]`. Click it. The kingdom note opens. | **VO:** "Wikilinks between every note." |
| 3–7s | Cut to backlinks panel — show "Mentioned in: Chapter 3, Elara, House Vael". Click one to jump. | **VO:** "Backlinks tell you who references what." |
| 7–13s | Cut to **Canvas**. A plot board: sticky notes for chapters, connectors showing arcs, shapes grouping acts. Drag a new chapter sticky in. Connect it. | **VO:** "Then plot it visually on an infinite canvas." |
| 13–18s | Drag the `Elara.md` doc from the sidebar onto the canvas — it pins as a doc node. Draw a connector from Elara → Chapter 3. | **VO:** "Pin any note onto the canvas. Reference canvas nodes from inside any doc." |
| 18–22s | Cut to **Focus Mode** (full screen, distraction-free editor). Type a sentence of prose. | **VO:** "Then focus mode when it's time to actually write." |
| 22–27s | Show the workspace folder in Finder / Explorer — `.md` files visible. | **VO:** "Everything's just Markdown in a folder you own. No cloud. No login." |
| 27–30s | End card: DevBoard logo + "free + Ko-fi link in bio" + 🇪🇺. | **VO:** "DevBoard. Link in bio." |
