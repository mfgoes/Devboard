# 15s Teaser — General audience

**Goal:** stop the scroll, plant the name "DevBoard" + the one-liner, drive to bio.
**Best for:** TikTok, Reels, Shorts. Use trending sound, no voiceover.
**Hook style:** visual + on-screen text only.

---

## Beats

| Time | On screen | Caption (overlay) |
|---|---|---|
| 0.0–1.5s | Cold open: app already in **Canvas mode**. Zoom out from a single sticky note to reveal a full plotted canvas with sticky notes + connectors + a code block. | **"Notes that actually think with you."** |
| 1.5–4.0s | Cut to **Markdown editor**. Type a `[[wikilink]]`. Click it — the note opens. Click backlinks panel — show 2–3 backlinks. | **"Wikilinks. Backlinks. Built in."** |
| 4.0–7.0s | Cut back to canvas. Drag a doc from the sidebar **onto the canvas** (pin a doc node). Draw a connector from it to a sticky note. | **"Notes ↔ canvas. Same workspace."** |
| 7.0–10.0s | Cut to **file explorer** sidebar. Open the workspace folder in the system file manager — show the actual `.md` files on disk. | **"Your folder. Your files. No cloud."** |
| 10.0–13.0s | Cut to landing page or app open in browser. Pan over the URL bar. | **"Free. Local-first. EU-built."** |
| 13.0–15.0s | End card: DevBoard logo + "free in your browser" + Ko-fi handle. | **"DevBoard → link in bio ❤️"** |

## Screens to record before filming

1. A canvas pre-built with ~8 sticky notes, 3 shapes, 2 connectors, 1 code block.
2. Two notes with mutual `[[wikilinks]]` so backlinks panel populates.
3. A workspace folder visible in your OS file manager (Finder / Explorer / Files).
