# DevBoard — Marketing Scripts

Scripts for promoting DevBoard on Ko-fi, Bluesky, X, and TikTok / Reels / Shorts.

## Core message (use this everywhere)

> **DevBoard** — a local-first notes + canvas app. Your folder, your files, no account, no cloud. EU-built, free, and open. Like Obsidian — but the infinite canvas is built in.

**One-liners** (pick what fits the platform):

## Pillars to lean on (in order of strength)

1. **Local-first / privacy** — files live in a folder you choose, plain Markdown + JSON. Open them in VS Code. Version with git. No login, no cloud, no telemetry.
2. **Notes AND canvas in one workspace** — wikilinks AND an infinite whiteboard, cross-referenced (`@node:` mentions, pin docs onto canvas).
3. **EU-built** — alternative to US SaaS note tools. Resonates strongly on Bluesky / Mastodon-adjacent crowds.
4. **Free + Ko-fi supported** — no subscription, no "pro tier" wall.

## Recording tips (read once before filming)

- **Workspace prep**: open a workspace folder pre-populated with realistic notes. Don't film an empty app — it reads as "demo bait." Pre-fill with 6–10 notes and at least one canvas with sticky notes + connectors.
- **Theme**: dark mode reads better in short-form video. Pick one accent color and stick to it across all clips.
- **Cursor**: bump the system cursor size up one notch — phone viewers can't track a tiny cursor.
- **Resolution**: record vertical 1080×1920 by cropping a windowed app, OR record landscape 1920×1080 and crop in post. A square 1:1 crop also works for Bluesky.
- **No audio narration on TikTok**: trending-sound + on-screen captions outperforms voiceover on cold reach. Save voiceover for the 60s+ scripts and YouTube.
- **First 1.5 seconds**: must show the app doing something visual (a wikilink resolving, a canvas zooming, a connector being drawn). Static text loses the scroll.
